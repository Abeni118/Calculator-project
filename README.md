# Simple JavaScript Calculator

### A basic calculator built with HTML, CSS, and JavaScript. It supports addition, subtraction, multiplication, division, decimals, and clearing the display.

## Features

- Perform basic arithmetic operations: +, -, *, /
- Support for decimal numbers
- Clear/reset functionality
- Chained calculations (e.g., 5 + 3 * 2)
- Responsive and simple user interface

## How to Use

1. Click on the number buttons to enter values.
2. Click an operator (+, -, *, /) to perform calculations.
3. Press = to get the result.
4.  Press C to reset the calculator.

## Code Highlights

- calculate object handles operations dynamically.
- awaitingNextValue flag ensures proper number input after operators.
- Event listeners handle number, operator, decimal, and clear buttons.
